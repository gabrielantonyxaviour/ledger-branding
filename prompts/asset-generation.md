# Ledger D1 — ChatGPT image-gen prompts (Auction House direction)

12 paste-ready prompts. Each is dense, repeats critical constraints inline so the model can't drift, and uses concrete reference language (Sotheby's catalogue, Christie's plate) instead of jargon.

**Universal constraints — every image must respect these:**
- Vector aesthetic, perfectly flat
- NO photorealism, NO 3D, NO gradients, NO drop shadows, NO glow, NO bloom, NO sparkles, NO emojis
- NO crypto cliches (no cubes, no rocket ships, no robots, no brain motifs)
- Numbers in serif; words in grotesque, religiously
- Tabular figures where applicable
- One bold typographic moment per image

**Palette reference:** paper `#F5F1E8` · ink `#0F0F12` · oxblood `#9C2A2A` · gold leaf `#C8A85C`

## Run order (sequential — same ChatGPT browser session)

1. Drop each prompt into a NEW chat (one chat per prompt to avoid context cross-pollution).
2. Wait ~30-60s per chat for the image to render.
3. Save the chat hrefs.
4. Run a separate download pass that pulls each `estuary/content` URL via authenticated `context.request.get()`.

## Iteration tips if something misfires

- **Wrong serif (got slab/sans/script)** → append: *"It MUST be a high-contrast modern italic serif like Bodoni Italic or Editorial New Italic — NOT a slab serif, NOT a sans-serif, NOT a script."*
- **Wrong background (got white/dark)** → restate at start: *"BACKGROUND: cream paper #F5F1E8. Not white. Not navy. Not black. The exact warm cream of a Sotheby's catalogue cover."*
- **Glow / bloom / 3D crept in** → append: *"Pure flat vector. Zero depth. Zero glow. Like ink printed on paper."*
- **Too much text / wrong text** → restate exact text and add: *"NO other text, NO sub-tagline, NO date, NO author byline, NO additional words besides what is specified."*
